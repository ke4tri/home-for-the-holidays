# home-for-the-holidays

### Description
This project creates an app that organizes the holiday parties one would have throughout the year with the friends who are invided.

### Technologies Used
* SASS
* Webpack
* Axios
* ES6 Modules

### How to run this app
1.  Serve up the app
* At the root of the project: `npm install`
* `npm start`
